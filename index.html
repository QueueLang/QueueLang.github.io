Introduction

QueueLang is a versatile interpreted programming language written in C++ for performance and speed. QueueLang is relatively stable and easy to use. QueueLang has all the basic needs, like variable support, user-defined and core-functions. It also supports more complicated things like math, lambdas, and plugin support.

Basic Syntax

queue x = 0
queue y = 5

console.out(toString(x + y))

Key Features

  • Interpreted language for quick execution and debugging.
  • Written in C++ for efficiency.
  • Supports basic arithmetics, necessary functions, lambdas, and plugin integration.

Examples

func add(x, y) {
    return x + y
}

console.out(toString(add(6, 3)))
