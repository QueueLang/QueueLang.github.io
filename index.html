Introduction

QueueLang is a versatile interpreted programming language written in C++. Designed for simplicity and speed, it features a queue-inspired syntax while supporting advanced constructs like functions and lambdas.

Basic Syntax

The following demonstrates the basics of QueueLang syntax:


queue x = 0
queue y = 5

console.out(toString(x + y))

Key Features

  • Interpreted language for quick execution and debugging.
  • Written in C++ for efficiency.
  • Supports functions, lambdas, and plugin integration.
  • Inspired by Python, with a unique focus on queues and functional programming.
